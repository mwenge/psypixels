psychedelia syndrome

Psychedelia was the first ever interactive 'light synthesizer', written for the Commodore 64 by Jeff Minter and published by Llamasoft in 1984.

psychedelia syndrome is an exploration of the game's code and an atlas of the pixels and effects it generated. It is a book about assembly code and how it works. It should appeal to anyone with an interest in computers, old or new, and provide interesting insights into how 40-year old software produced appealing graphical displays. The book provides a complete, academic-style commentary on the code for all editions of Psychedelia and is leavened throughout with pretty pictures.

You can download and read the book here. A dual-page view in your PDF reader is recommended.

The electronic edition is free, but if you like it you can gift what you want.
